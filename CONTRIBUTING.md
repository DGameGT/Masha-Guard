# Contributing to MashaGuard

MashaGuard is a closed-source project. This repository does not accept Pull Requests from external contributors.

---

## Joining as a Developer

If you are interested in contributing to MashaGuard as an official team member, you may submit an application to:

**contact@dgxo.my.id**

Please include the following in your email:
- A brief introduction and your technical background
- The area you would like to contribute to (backend, SDK, forensics, dashboard, etc.)
- Links to your portfolio, GitHub, or any relevant work

We review every application and will follow up if there is a fit.

---

## Reporting Bugs or Vulnerabilities

For general bugs, open an issue in this repository with a clear description and reproduction steps.

For security vulnerabilities, do not open a public issue. Refer to [SECURITY.md](./SECURITY.md) for the correct reporting procedure.
